			==============================================================
				Reading the Data OR Input Dynamically from Key Board
			==============================================================
=>To Read the Data OR Input Dynamically from Key Board to the Python Program, we use Two Pre-Defined Functions. They are
				1. input()
				2. input(Message)
------------------------------------------------------------------------------------------------------------------------------------------------------
1. input()
------------------------------------------------------------------------------------------------------------------------------------------------------
=>Syntax:    Varname=input()
=>This function is used reading  any type of Value from key Board  in the form of str and placed in LHS Varname.
=>here str value can be converted into Desired type by using Type Casting Functions.
------------------------------------------------------------------------------------------------------------------------------------------------------
2. input(Message)
------------------------------------------------------------------------------------------------------------------------------------------------------
=>Syntax:    Varname=input(Message)
=>here Message Represents user-Prompting Message.
=>This function is used reading  any type of Value from key Board  in the form of str and placed in LHS Varname and along this It can display User-Prompting Message.
=>here str value can be converted into Desired type by using Type Casting Functions.
------------------------------------------------------------------------------------------------------------------------------------------------------
