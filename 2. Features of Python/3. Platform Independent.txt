Def. of Platform:
-------------------------
=>Type of OS Being Used to Run OR Execute the Application OR Program OR Project is called Platform.
--------------------------------------------------------
Def. of OS(Operating System)
--------------------------------------------------------
=>OS is a Software which acts as Resources Allocation and De-allocation Manager.
	Here Resources are Nothing Computer Hardware--->RAM, Hard Disk, CPU, Moniter, KeyBoard...etc
					OR
=>OS is one the Interface between Program and Computer hardware.

1. Paltform Dependent Languages
