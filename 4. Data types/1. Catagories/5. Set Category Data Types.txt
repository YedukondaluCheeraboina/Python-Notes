			==============================================================
							 Set Category Data Types
			==============================================================
=>The purpose of Set Category Data Types is that "To store Multiple values either of Same type or Different type or Both the Types in single Object with UniqueValues ( No Duplicates are allowed) ".
=>We have 2 Data types in Set Category. They are
					1. set ( Mutable and Immutable)
					2. frozenset (Immutable)
