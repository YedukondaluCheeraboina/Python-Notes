				=============================================================
								1. int()
				=============================================================
=>Here int() is used for converting Possible Type of Values into int type.
=>Syntax:   varname=int(float / bool / complex / str )
-------------------------------------------------------------------------------------------------------------------------------------------------------
Example-1:   float type into int type----Possible
-------------------------------------------------------------------------------------------------------------------------------------------------------
>>> a=12.34
>>> print(a,type(a))
12.34 <class 'float'>
>>> b=int(a)
>>> print(b,type(b))
12 <class 'int'>
-------------------------------------------------------------------------------------------------------------------------------------------------------
Example-2:   bool type into int type--Possible
-------------------------------------------------------------------------------------------------------------------------------------------------------
>>> a=True
>>> print(a,type(a))
True <class 'bool'>
>>> b=int(a)
>>> print(b,type(b))
1 <class 'int'>
>>> b=int(False)
>>> print(b,type(b))
0 <class 'int'>
-------------------------------------------------------------------------------------------------------------------------------------------------------
Example-3:  complex type into int type------Not Possible
-------------------------------------------------------------------------------------------------------------------------------------------------------
>>> a=2+3j
>>> print(a,type(a))-------------------(2+3j) <class 'complex'>
>>> b=int(a)----------TypeError: int() argument must be a string
>>> x=int(a.real)
>>> print(x)-----------------2
>>> y=int(a.imag)
>>> print(y)----------------3
-------------------------------------------------------------------------------------------------------------------------------------------------------
Example-4:   str type into int type
-------------------------------------------------------------------------------------------------------------------------------------------------------
Case-1:  str int into  int--Possible
-------------------------------------------------
>>> a="12"
>>> print(a,type(a))
12 <class 'str'>
>>> b=int(a)
>>> print(b,type(b))
12 <class 'int'>
-------------------------------------------------
Case-2:  str float into  int------Not Possible
-------------------------------------------------
>>> a="12.34"
>>> print(a,type(a))---------------------12.34 <class 'str'>
>>> b=int(a)-------------------------------ValueError: invalid literal for int() with base 10: '12.34'
-------------------------------------------------
Case-3:  str bool into  int--Not Possible
-------------------------------------------------
>>> a="True"
>>> print(a,type(a))-----------------True <class 'str'>
>>> b=int(a)---------------------------ValueError: invalid literal for int() with base 10: 'True'
-------------------------------------------------
Case-4:  str complex into  int-----Not Possible
-------------------------------------------------
>>> a="2+3.5j"
>>> print(a,type(a))--------------------2+3.5j <class 'str'>
>>> b=int(a)------------------------------ValueError: invalid literal for int() with base 10: '2+3.5j'
-------------------------------------------------
Case-5:  Pure str  into  int-----Not Possible
-------------------------------------------------
>>> a="Python"
>>> print(a,type(a))---------------------Python <class 'str'>
>>> b=int(a)-------------------------------ValueError: invalid literal for int() with base 10: 'Python'
>>> b=int("10x")-------------------------ValueError: invalid literal for int() with base 10: '10x'
