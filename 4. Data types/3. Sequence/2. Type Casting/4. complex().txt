				=============================================================
								4. complex()
				=============================================================
=>Here complex() is used for converting Possible Type of Values into complex type.
=>Syntax:   varname=complex(int / float/ bool / str )
-------------------------------------------------------------------------------------------------------------------------------------------------------
Example-1:   int type into complex type--Possible
-------------------------------------------------------------------------------------------------------------------------------------------------------
>>> a=10
>>> print(a,type(a))
10 <class 'int'>
>>> b=complex(a)
>>> print(b,type(b))
(10+0j) <class 'complex'>
-------------------------------------------------------------------------------------------------------------------------------------------------------
Example-2:   float type into complex type--Possible
-------------------------------------------------------------------------------------------------------------------------------------------------------
>>> a=10.5
>>> print(a,type(a))
10.5 <class 'float'>
>>> b=complex(a)
>>> print(b,type(b))
(10.5+0j) <class 'complex'>
------------------------------------------------------------------------------------------------------------------------------------------------------
Example-3:  bool type  into complex type --Possible
-------------------------------------------------------------------------------------------------------------------------------------------------------
>>> a=True
>>> print(a,type(a))
True <class 'bool'>
>>> b=complex(a)
>>> print(b,type(b))
(1+0j) <class 'complex'>
>>> b=complex(False)
>>> print(b,type(b))
0j <class 'complex'>
-------------------------------------------------------------------------------------------------------------------------------------------------------
Example-4:   str type into complex type
-------------------------------------------------------------------------------------------------------------------------------------------------------
Case-1:  str int into  complex--Possible
-------------------------------------------------
>>> a="12"
>>> print(a,type(a))
12 <class 'str'>
>>> b=complex(a)
>>> print(b,type(b))
(12+0j) <class 'complex'>
-------------------------------------------------
Case-2:  str float into  compelx--Possible
-------------------------------------------------
>>> a="3.5"
>>> print(a,type(a))
3.5 <class 'str'>
>>> b=complex(a)
>>> print(b,type(b))
(3.5+0j) <class 'complex'>
------------------------------------------------
Case-3:  str bool into  complex--Not Possible
-------------------------------------------------
>>> a="True"
>>> print(a,type(a))
True <class 'str'>
>>> b=complex(a)-----------------ValueError: complex() arg is a malformed string
-------------------------------------------------
Case-4:  str complex into  complex--Possisble
-------------------------------------------------
>>> a="2+3.5j"
>>> print(a,type(a))
2+3.5j <class 'str'>
>>>
>>> b=complex(a)
>>> print(b,type(b))
(2+3.5j) <class 'complex'>
-------------------------------------------------
Case-5:  Pure str  into  complex--Not Possible
-------------------------------------------------
>>> a="PYTHON"
